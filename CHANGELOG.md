# سجل التغييرات - Changelog

## [2.0.0] - 2026-02-20

### 🎉 إصدار كامل - Full Release

#### ✨ الميزات الجديدة

**نقطة البيع (POS)**
- إضافة 18 اختصار قابل للتخصيص للمنتجات
- دعم النقر بالزر الأيمن لتعديل الاختصارات
- إضافة 20 لون للاختصارات
- حفظ الاختصارات في localStorage
- عرض اسم المنتج والسعر في صندوق الاختصار

**إدارة الحصص اليومية**
- فتح/غلق الحصة مباشرة من صفحة POS
- شريط معلومات الجلسة (كود، نقدي، دين، إجمالي، عدد العمليات)
- نافذة تفاصيل الحصة
- نافذة تأكيد إغلاق الحصة

**نظام الأخطاء (System Errors)**
- API جديد لإدارة أخطاء النظام
- 7 endpoints للـ CRUD والصيانة
- واجهة مستخدم متصلة بـ API حقيقي
- بطاقات إحصائيات (إجمالي، حرجة، تحذيرات، معلومات، محلولة، اليوم)
- إجراءات الصيانة السريعة (مسح الكاش، إعادة الاتصال، إعادة التشغيل، فحص النظام)
- إصلاح تلقائي للأخطاء القابلة للإصلاح

**النسخ الاحتياطي**
- نظام نسخ احتياطي متاح لكل مستأجر
- تصدير JSON/CSV
- استيراد من JSON
- إعدادات النسخ التلقائي (يومي/أسبوعي/شهري)

**استيراد قواعد البيانات**
- دعم استيراد من Microsoft Access (.mdb/.accdb)
- تحويل تلقائي إلى JSON
- واجهة استيراد في لوحة تحكم SaaS Admin

#### 🐛 إصلاح الأخطاء
- إصلاح CashBoxResponse schema (جعل updated_at اختياري)
- إصلاح عرض سعر المنتج في اختصارات POS
- إصلاح مشاكل الـ RTL في بعض الصفحات

#### 🔧 تحسينات
- تحسين أداء تحميل المنتجات
- تحسين استجابة واجهة POS
- تنظيف الكود وحذف الملفات غير المستخدمة
