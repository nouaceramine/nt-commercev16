## 1. سوبر أدمين (Super Admin)

### الوصف
المسؤول الأعلى عن نظام NT Commerce بالكامل. يدير الوكلاء والخطط والتقارير المالية الشاملة.

### الصلاحيات
| الصلاحية | الوصف |
|----------|-------|
| إدارة الوكلاء | إضافة/تعديل/حذف/تفعيل/تعطيل الوكلاء |
| إدارة الخطط | إنشاء وتعديل خطط الاشتراك والأسعار |
| التقارير المالية | عرض جميع الإيرادات والعمولات |
| إدارة المشتركين | عرض جميع المشتركين وتمديد اشتراكاتهم |
| إعدادات النظام | التحكم الكامل في إعدادات النظام |
| تحديثات النظام | إرسال إعلانات لجميع المشتركين |

### لوحة التحكم
- المسار: `/saas-admin`
- الميزات:
  - إحصائيات شاملة (وكلاء، مشتركين، إيرادات)
  - إدارة الوكلاء
  - إدارة المشتركين
  - إدارة الخطط
  - التقارير المالية
  - سجل المدفوعات

## 2. الوكيل (Agent)

### الوصف
موزع/ريسيلر يقوم ببيع اشتراكات NT Commerce. يحصل على عمولة عن كل مشترك.

### الصلاحيات
| الصلاحية | الوصف |
|----------|-------|
| إضافة مشتركين | تسجيل مشتركين جدد تحت حسابه |
| تعديل المشتركين | تعديل بيانات مشتركيه |
| تجديد الاشتراكات | تجديد اشتراكات مشتركيه |
| عرض العمولات | تتبع عمولاته ومعاملاته المالية |
| عرض الرصيد | تتبع رصيده وحد الدين |

### لوحة التحكم
- المسار: `/agent/dashboard`
- الميزات:
  - الرصيد المالي (الرصيد الحالي، حد الدين، إجمالي العمولات)
  - إحصائيات المشتركين (إجمالي، نشط، تجريبي)
  - قائمة المشتركين
  - إضافة مشترك جديد
  - سجل المعاملات

### البيانات المالية
```
الرصيد الحالي = المدفوعات - (قيمة الاشتراكات - العمولات)
العمولة = قيمة الاشتراك × نسبة العمولة
        أو عمولة ثابتة لكل اشتراك
```

## 3. المشترك (Tenant)

### الوصف
صاحب متجر/نشاط تجاري يستخدم نظام NT Commerce لإدارة مبيعاته ومخزونه.

### الصلاحيات
| الصلاحية | الوصف |
|----------|-------|
| نظام POS الكامل | نقطة البيع والمبيعات |
| إدارة المنتجات | إضافة/تعديل/حذف المنتجات |
| إدارة المخزون | الجرد والمخازن والتحويلات |
| إدارة الزبائن | العملاء والديون |
| إدارة الموردين | المشتريات والمدفوعات |
| إدارة الموظفين | إضافة موظفين بأدوار مختلفة |
| التقارير | تقارير المبيعات والمخزون والمالية |
| الإعدادات | إعدادات المتجر |

### قاعدة البيانات (Multi-tenancy) ✅ مفعّل
**كل مشترك له قاعدة بيانات منفصلة تماماً:**

```
اسم قاعدة البيانات: tenant_{tenant_id}
مثال: tenant_8f714452_ebe0_465f_a5bf_34f1d4f083f4
```

**المجموعات التي يتم إنشاؤها تلقائياً:**
- `users` - مستخدمي المشترك
- `settings` - إعدادات المتجر
- `warehouses` - المخازن
- `cash_boxes` - صناديق النقد
- `products` - المنتجات (تُضاف لاحقاً)
- `customers` - الزبائن (تُضاف لاحقاً)
- `suppliers` - الموردين (تُضاف لاحقاً)
- `sales` - المبيعات (تُضاف لاحقاً)
- `purchases` - المشتريات (تُضاف لاحقاً)

### لوحة التحكم
- المسار: `/tenant/dashboard` → `/` (النظام الرئيسي)
- الوصول الكامل لنظام POS

## 4. موظفي المشترك (Tenant Users)

### الأدوار المتاحة

| الدور | الوصف بالعربية | الصلاحيات الرئيسية |
|-------|---------------|-------------------|
| `admin` | مدير | صلاحيات كاملة على المتجر |
| `manager` | مشرف | إدارة العمليات اليومية |
| `sales_supervisor` | مشرف مبيعات | إشراف على المبيعات والعملاء |
| `seller` | بائع | عمليات البيع الأساسية فقط |
| `inventory_manager` | مدير مخزون | إدارة المخزون والمشتريات |
| `ecommerce_manager` | مسؤول متجر إلكتروني | إدارة المتجر الإلكتروني |
| `accountant` | محاسب | التقارير المالية والديون والمصاريف |
| `user` | مستخدم | عرض فقط |

### تفصيل الصلاحيات

#### مدير (admin)
- ✅ كل الصلاحيات على المتجر
- ✅ إدارة المستخدمين
- ✅ الإعدادات
- ❌ إدارة SaaS

#### مشرف (manager)
- ✅ نقطة البيع
- ✅ المنتجات (عرض، إضافة، تعديل)
- ✅ المبيعات والمرتجعات
- ✅ الزبائن والديون
- ✅ التقارير الأساسية
- ❌ حذف البيانات
- ❌ تغيير الأسعار
- ❌ إدارة المستخدمين

#### بائع (seller)
- ✅ نقطة البيع
- ✅ إضافة مبيعات
- ✅ عرض المنتجات والزبائن
- ❌ التعديل والحذف
- ❌ التقارير
- ❌ المشتريات

#### محاسب (accountant)
- ✅ التقارير المالية الكاملة
- ✅ إدارة المصاريف
- ✅ تحصيل الديون
- ✅ مدفوعات الموردين
- ❌ نقطة البيع
- ❌ تعديل المنتجات

#### مدير مخزون (inventory_manager)
- ✅ إدارة المنتجات
- ✅ الجرد والتحويلات
- ✅ المشتريات
- ✅ الموردين
- ❌ نقطة البيع
- ❌ المبيعات

## APIs الرئيسية

### تسجيل الدخول
```
POST /api/auth/unified-login
Body: { "email": "string", "password": "string" }
Response: {
  "access_token": "string",
  "user_type": "admin" | "agent" | "tenant",
  "redirect_to": "/path",
  "user": { ... }
}
```

### إدارة الوكلاء (سوبر أدمين فقط)
```
GET    /api/saas/agents          - قائمة الوكلاء
POST   /api/saas/agents          - إضافة وكيل
PUT    /api/saas/agents/{id}     - تعديل وكيل
DELETE /api/saas/agents/{id}     - حذف وكيل
```

### إدارة المشتركين
```
GET    /api/saas/tenants              - قائمة المشتركين
POST   /api/saas/tenants              - إضافة مشترك
GET    /api/tenants/by-agent/{id}     - مشتركين الوكيل
PUT    /api/saas/tenants/{id}         - تعديل مشترك
```

### إدارة الخطط
```
GET    /api/saas/plans           - قائمة الخطط
POST   /api/saas/plans           - إضافة خطة
PUT    /api/saas/plans/{id}      - تعديل خطة
```
